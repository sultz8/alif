## 1. Создайте свою папку:

```
/path/to/your/folder/alif
```

## 2. Настройте файл окружения:

```
.env.example -> .env
```

## 3. Запуск:

```
docker-compose up -d
```

## Данные для входа в базу данных указаны в .env.example

## API:
```
http://localhost:8876/api
```

## Вход в контейнер:
```
docker exec -it php-fpm-1 bash
```
